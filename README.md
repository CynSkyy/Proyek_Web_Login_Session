==Panduan Sederhana Penggunaan Project Restoran Nusantara==

- Halaman Home ada pada file (Project.php)
- Halaman-halaman berikutnya menggunakan file berformat (HTML/PHP)
- Folder (auth) digunakan untuk menyimpan fitur (Login dan Register)
- Folder (admin) digunakan untuk mengakses data user
- Semua file (CSS dan gambar) berada di folder (assets)
- Konfigurasi database terdapat pada file (config/database.php)
- File SQL database terdapat pada folder (database) dengan nama (restoran.sql)

==Panduan Penggunaan Web==

- Masuk ke halaman (Project.php)
- Klik tombol (ORDER DI SINI)
- Lakukan (Registrasi) jika belum memiliki akun
- (Login) menggunakan email dan password yang sudah terdaftar
- Setelah login:

  - (User) akan masuk ke halaman menu makanan
  - Pilih menu dengan menekan tombol (Pesan)
  - Menu akan masuk ke (keranjang)
  - Buka halaman keranjang lalu klik (Checkout)
  - Jika berhasil, akan muncul halaman (pesanan berhasil)
- Admin yang login hanya dapat melihat data user yang terdaftar
- Untuk keluar dari sistem, klik tombol **Logout**
